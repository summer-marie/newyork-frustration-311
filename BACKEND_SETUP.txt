✅ API ENDPOINTS (Running on http://localhost:3001)
---------------------------------------------------

Noise Endpoints:
  GET /api/noise/by-borough       → Noise complaints by NYC borough
  GET /api/noise/by-hour          → Noise complaints by hour (0-23)
  GET /api/noise/by-zip           → Top 20 ZIPs by noise volume
  GET /api/noise/by-type          → Breakdown by complaint type

Rodent Endpoints:
  GET /api/rodent/by-borough      → Rodent complaints by NYC borough
  GET /api/rodent/by-zip          → Top 20 ZIPs by rodent volume

Analysis Endpoints:
  GET /api/overlap/zip            → ZIPs in both top-20 lists

✅ PACKAGE.JSON SCRIPTS
-----------------------

npm run server       → Start Express API server (port 3001)
npm run dev:server   → Same as above (alias)
npm run dev          → Start Vite dev server (port 5173)


✅ HOW TO RUN BOTH SERVERS
--------------------------

Terminal 1:  npm run server
Terminal 2:  npm run dev

The frontend (port 5173) can now fetch from /api/... 
and Vite will proxy to the Express backend (port 3001).


✅ EXAMPLE USAGE IN FRONTEND
----------------------------

// Fetch noise data from API instead of importing JSON
async function loadNoiseData() {
  const response = await fetch('/api/noise/by-borough')
  const data = await response.json()
  initNoiseByBoroughChart(data)
}

// Fetch overlap data
async function loadOverlapData() {
  const response = await fetch('/api/overlap/zip')
  const data = await response.json()
  initZipOverlapTable(data)
}
